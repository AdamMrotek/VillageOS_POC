VillageOS — Data Schema

Adam Mrotek · 2026-05-04

Overview

VillageOS is a family operating system for parents. It turns the chaotic stream of school communications — emails, newsletters, WhatsApp threads — into a single, structured calendar a parent can actually act on.

AI Extraction

Paste any unstructured text — PDF newsletters, forwarded emails, WhatsApp threads. The AI layer produces a structured event (title, date, location, action items with costs) for parent review. No manual data entry.

RAG Discovery

Natural-language search over local providers and stored events. Vector similarity search with metadata filters surfaces relevant results; an LLM synthesises a short, specific recommendation explaining the match.

Data Flows

AI Extraction — LLM converts unstructured text into structured events
RAG Discovery — vector search + LLM synthesis over provider data
